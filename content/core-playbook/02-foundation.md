## 2.1 Vision Alignment: The Exercise Nobody Does

Most couples skip straight to Pinterest boards and venue tours. The ones who do this spend the first three months of planning in a low-grade disagreement about aesthetic, scale, and tone — without ever naming it as such.

Vision alignment is a 30-minute exercise. It prevents months of friction.

[Image: two people at a kitchen table, each with a notepad, comparing notes — warm light, casual setting]

### The Vision Alignment Exercise

Do this independently first. Then compare.

**Part 1: The Word List (5 minutes each)**

Write 5–10 words that describe how you want the day to FEEL. Not look — feel.

Examples: _intimate, joyful, relaxed, elegant, rowdy, romantic, earthy, dramatic, celebratory, quiet, grand_

Compare your lists. Circle the overlaps. These are your anchors.

**Part 2: The "What We're Not" List (5 minutes each)**

Write 3–5 descriptors for weddings you have attended or seen that felt wrong to you — or that are explicitly NOT what you want.

Examples: _overly formal, stressful, rushed, expensive-for-the-sake-of-it, performative, generic, chaotic_

Compare. This is your filter — when a vendor or upgrade pulls you toward one of these, you have permission to say no.

**Part 3: The Non-Negotiable 3 (10 minutes together)**

Identify the three things that MUST be excellent. Everything else can be good enough.

| Partner A's Top 3 | Partner B's Top 3 | Agreed Top 3 |
| ----------------- | ----------------- | ------------ |
|                   |                   |              |
|                   |                   |              |
|                   |                   |              |

**Part 4: The "Guests Will Remember" Statement**

Complete this sentence together:

> "When guests leave our wedding, we want them to feel ****\_\_\_****."

One sentence. Write it down. Put it where you plan. Every major decision gets held against this sentence.

### Vision Red Flags — Check These

- [ ] One partner is significantly more invested in the planning than the other — this creates resentment
- [ ] You are planning toward what looks good in photos vs. what feels right in person
- [ ] You have not aligned on formality level (black tie vs. garden party changes everything downstream)
- [ ] You have not aligned on guest count philosophy (intimate = 50, medium = 100, large = 150+)

---

> **Foundation Rule #1:** Design for how the day will feel to the two of you. Everything else is set dressing.

## 2.2 Budget Architecture: Setting the Number That Holds

Setting a wedding budget is not about choosing a number. It is about choosing a number that is financially sustainable AND emotionally defensible — one you can repeat to vendors, family, and yourselves without flinching.

[Image: a budget architecture diagram — income sources at top feeding into a single "Wedding Budget Pool," then allocated across categories]

### Step 1: Identify All Income Sources

Be honest here. Unrealized contributions are not budget.

| Source                               | Confirmed Amount | Conditions Attached     |
| ------------------------------------ | ---------------- | ----------------------- |
| Your savings                         | $                | None                    |
| Partner's savings                    | $                | None                    |
| Joint savings                        | $                | None                    |
| Parent contribution (your side)      | $                | Describe any conditions |
| Parent contribution (partner's side) | $                | Describe any conditions |
| Other gifts/contributions            | $                | Describe any conditions |
| **TOTAL AVAILABLE**                  | **$**            |                         |

**About parent contributions:** Money with conditions is not the same as unconditional money. If accepting $5,000 from a parent means they have significant input on guest list, venue, or aesthetic — that contribution may cost more than it's worth in stress and conflict. Have an explicit conversation before counting it.

### Step 2: Set the Ceiling

Take your total available amount. Subtract 10% for contingency. The remaining number is your **spending ceiling** — not target, ceiling.

> Total Available: $**\_\_** — Contingency (10%): $**\_\_** = **Spending Ceiling: $**\_\_****

### Step 3: Allocate by Category

Use these percentages as starting points. Adjust based on your Non-Negotiable Top 3 — weight those categories heavier, cut from the rest.

| Category                   | Standard % | Your Amount |
| -------------------------- | ---------- | ----------- |
| Venue + Catering           | 35%        | $           |
| Photography + Video        | 14%        | $           |
| Florals + Décor            | 9%         | $           |
| Music                      | 7%         | $           |
| Attire + Alterations       | 6%         | $           |
| Hair + Makeup              | 3%         | $           |
| Stationery + Paper         | 2%         | $           |
| Transportation             | 2%         | $           |
| Rehearsal Dinner           | 5%         | $           |
| Officiant + Ceremony       | 2%         | $           |
| Favors + Gifts             | 2%         | $           |
| Miscellaneous + Tips       | 3%         | $           |
| **Contingency (reserved)** | **10%**    | **$**       |

### Step 4: The Reality Check

Run these three numbers before you commit to your allocation:

**Cost per head** = Spending ceiling ÷ Guest count

- Under $175/person: You will be cutting corners that guests notice
- $200–$275/person: Comfortable range for seated dinner + open bar
- $300+/person: Premium experience is achievable

**Photography share** = Photography budget ÷ Total budget

- Under 10%: You will likely compromise on quality
- 12–15%: Industry sweet spot for quality documentation

**Venue-to-total ratio** = Venue budget ÷ Total budget

- Over 40%: Venue is consuming too much — adjust expectations or total budget

---

> **Foundation Rule #2:** If the cost-per-head math doesn't work, the only two variables are budget and guest count. Pick one to change.

## 2.5 Guest List Architecture

The guest list is the second-most consequential decision in wedding planning (after budget). It directly determines your venue options, catering cost, logistics complexity, and overall intimate-to-large-event feel. It also generates more interpersonal conflict than any other single planning element.

[Image: three concentric circles — innermost "Must Have" (A-list), middle "Love to Have" (B-list), outer "Obligation" (C-list)]

### Building the List in 5 Steps

**Step 1: Independent Brainstorm**
Each partner separately lists every person they could imagine inviting. No editing, no filtering — just capture. This takes 30 minutes and should happen before any conversation with family.

**Step 2: Joint Review — A/B/C Sort**
Go through the combined list together. Sort each name:

- **A-List:** Would genuinely hurt not to have them there
- **B-List:** Would love to include if capacity allows
- **C-List:** Obligation invite — invite knowing they likely can't attend

**Step 3: Set the Ceiling by Category**
Before families get involved, allocate your total guest slots:

| Category                                | Your Allocation | Partner's Allocation |
| --------------------------------------- | --------------- | -------------------- |
| Immediate family                        |                 |                      |
| Extended family                         |                 |                      |
| Close friends                           |                 |                      |
| Work/social connections                 |                 |                      |
| Family allocation (each side's parents) |                 |                      |
| **Total**                               |                 |                      |

**Step 4: Parent Conversation (With Your Numbers in Hand)**
Tell each set of parents: "We have X total spots. We're allocating Y to each side's family. Here are yours to fill how you choose."

Do not ask for their list before you have set the ceiling. The list will always be larger than the ceiling allows.

**Step 5: The B-List Protocol**
When A-list RSVPs come in and you have space, work the B-list from top to bottom. Mail the next round of invitations 6–8 weeks before the wedding. Most B-list guests never know — and those who do are rarely offended if handled warmly.

### Guest List Rules That Prevent Drama

| Rule                                                             | Why It Matters                                                            |
| ---------------------------------------------------------------- | ------------------------------------------------------------------------- |
| "All of a friend group or none"                                  | Selectively inviting part of a friend group creates lasting fractures     |
| "Plus-ones for partnered guests only"                            | Apply this consistently — no exceptions without updating the rule for all |
| "Children under 16 not invited" OR "Children welcome" — pick one | Mixed policies create conflict                                            |
| "No last-minute additions after invitations go out"              | Enforce this even under pressure                                          |
| Keep the B-list confidential                                     | Two B-listers comparing notes will figure it out                          |

### RSVP Tracking Template

| Name | Side | Invited | RSVP | Meal Choice | Dietary | Table # | Gift Noted |
| ---- | ---- | ------- | ---- | ----------- | ------- | ------- | ---------- |
|      |      |         |      |             |         |         |            |

Track every column from day one. The seating chart, caterer headcount, and place cards all flow from this sheet.

---

> **Foundation Rule #5:** Guest list drama is almost always caused by a missing rule, not a missing invitation. Set the rules before you set the list.

## 2.6 Venue Type Comparison

Choosing a venue type before you fall in love with a specific venue is a planning superpower. Different venue categories have dramatically different cost structures, vendor flexibility, logistics profiles, and aesthetic ranges.

[Image: 5 venue-type photos side by side — hotel ballroom, barn, vineyard, restaurant, backyard — each with a cost and complexity rating underneath]

### The 5 Major Venue Types

**Type 1: Hotel Ballroom / Banquet Hall**

_Best for:_ Large guest counts, formal aesthetic, out-of-town guest convenience
_Cost profile:_ Per-head pricing, often includes catering in-house
_Pros:_ Turnkey logistics, all-weather, built-in overnight accommodations, professional staff
_Cons:_ Generic aesthetic, limited vendor flexibility, early curfews, can feel impersonal
_Hidden costs to watch:_ Mandatory minimum spend, service charge (18–22%), cake cutting fee, parking

**Type 2: Historic Venue / Estate / Mansion**

_Best for:_ Couples who want architectural beauty without heavy décor investment
_Cost profile:_ Flat rental fee, bring-your-own vendors
_Pros:_ Stunning built-in aesthetic, often high flexibility, exclusive feel
_Cons:_ You source everything — catering, rentals, furniture, lighting
_Hidden costs to watch:_ Generator rental for outdoor areas, restroom trailer if insufficient facilities, load-in complexity

**Type 3: Barn / Farm / Rustic**

_Best for:_ Relaxed, warm, outdoor-adjacent vibe; often good value
_Cost profile:_ Flat rental, bring your own catering
_Pros:_ Instagram-friendly, relaxed atmosphere, often outdoor/indoor flexibility
_Cons:_ Weather dependency, bug/pest considerations, septic limitations, often remote
_Hidden costs to watch:_ Tent rental for weather backup, generator, transportation for guests

**Type 4: Restaurant / Private Dining**

_Best for:_ Intimate guest counts (30–80), food-forward couples
_Cost profile:_ Per-head or minimum spend
_Pros:_ Outstanding food quality, intimate scale, built-in service staff
_Cons:_ Limited to existing space layout, often can't customize heavily, guest count ceiling
_Hidden costs to watch:_ Corkage fee if bringing outside wine/spirits, AV limitations for music

**Type 5: Non-Traditional (Museum, Garden, Rooftop, Private Property)**

_Best for:_ Couples with a specific vision who are organized enough to source vendors independently
_Cost profile:_ Varies widely — rental fee or minimum spend
_Pros:_ Unique, memorable, highly customizable
_Cons:_ Often the most logistically complex — you are building the wedding from scratch
_Hidden costs to watch:_ Security deposits, event insurance requirements, permit costs, infrastructure rentals

### Venue Comparison Scorecard

| Factor                     | Venue 1 | Venue 2 | Venue 3 |
| -------------------------- | ------- | ------- | ------- |
| Base cost / per head       |         |         |         |
| All-in estimated total     |         |         |         |
| Vendor flexibility (1–5)   |         |         |         |
| Aesthetic match (1–5)      |         |         |         |
| Logistics complexity (1–5) |         |         |         |
| Weather risk (1–5)         |         |         |         |
| Guest experience (1–5)     |         |         |         |
| **Total Score**            |         |         |         |

---

> **Foundation Rule #6:** Fall in love with a venue type before you fall in love with a venue. It protects you from signing a beautiful contract you can't afford to staff.
