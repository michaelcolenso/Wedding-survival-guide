## Content Architecture

### Tone & Format Rules (confirmed across all phases)

- Warm big-sister voice, zero fluff, checklist-heavy
- Every section ends with a bold "Rule #N" callout in a blockquote
- Tables for scorecards, red-flag lists, comparisons
- [Image: description] placeholders throughout
- 420–450 words per page target

### Framework Conventions

- Section 1: "Survival Principles" (#1-6)
- Section 2: "Foundation Rules" (#1-6)
- Section 3: "Booking Rules" (#1-9)
- Section 4: "Execution Rules" (#1-8)
- Section 5: "Countdown Rules" (#1-6)
- Section 6: "Reception Rules" (#1-9)
